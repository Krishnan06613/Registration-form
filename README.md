# Responsive Registration Form

A modern and responsive Registration Form built using HTML, CSS, and JavaScript. This project includes real-time form validation and a clean user interface.

## Features

- Responsive Design
- Real-Time Input Validation
- Gmail Validation (@gmail.com)
- Password Minimum 10 Characters
- Register Button Enabled Only After Valid Inputs
- User-Friendly Interface
- Mobile Responsive

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Validation Rules

### Name
- Cannot be empty

### Email
- Must end with `@gmail.com`

### Password
- Minimum 10 characters

### Register Button
- Disabled until all fields are valid
